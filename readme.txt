Html website for a Portfolio website tags used are given below:

1. Head
2. Main
3. Footer
4. Hyperlink
5. Background Images
6. Navigation Bar
7. Div tag
8. Section
9. Body
10. Breakline
11. Unordered List


#  head tag-contains specific information about the web page.
#  Main tag-has the main content of the web page.
#  Footer tag-assign footer for a web page.
#  Hyperlink-link one page to another.
#  Background Image-to make the website attractive.
#  Navigation tag-represents section of a web page to navigate to same document or other document.
#  Division tag-to group similiar content together.
#  Section tag- to differentaite differentsections of the webpage.
#  Body tag-defines the body of html   web page.
#  Breakline- to write on the next line
#  Unordered list- to give bulleted list or dots.
